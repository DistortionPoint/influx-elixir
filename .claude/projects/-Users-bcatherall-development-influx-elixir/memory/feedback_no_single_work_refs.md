Do not reference single work projects or individual design documents in CLAUDE.md.

**Why:** CLAUDE.md is for standing project guidance and conventions, not pointers to specific work items.

**How to apply:** Keep CLAUDE.md focused on persistent project-level instructions. Reference design docs in conversation context or memory, not in CLAUDE.md.
